## Open-Source Contributions

Full list at [github.com/abuzarmahmood](https://github.com/abuzarmahmood)

**[blech_clust](https://github.com/abuzarmahmood/blech_clust)** — Spike-sorting library for multi-channel electrophysiological recordings featuring automated waveform classification and clustering, quality control, and data handling methods.

**[pytau](https://github.com/abuzarmahmood/pytau)** — Python package for Bayesian changepoint modelling with multiple inference algorithms (MCMC/ADVI), Bayesian nonparametric priors, and streamlined batch processing and model tracking. Published in [JOSS](https://doi.org/10.21105/joss.08509).

**[neuRecommend](https://github.com/abuzarmahmood/neuRecommend)** — XGBoost-based extracellular electrophysiology spike-waveform classifier.

**[Autoslide](https://github.com/abuzarmahmood/autoslide)** — Convolutional neural network-based semi-automated evaluation of perivascular fibrosis in histological sections.
